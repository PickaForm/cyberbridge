# CyberStreet Anti-Woke Carnage

A cyberpunk atmosphere street game with only boyz and girlz.
The project generates a procedural city (buildings, stands, crowd, flying traffic) and orchestrates real-time rendering with modular gameplay systems.

## Run in development mode

From the project root directory, run:

```bash
npm.cmd run dev
```

## Production build

```bash
npm.cmd run build
```
